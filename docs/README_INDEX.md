## üéØ Quick Links by Task

### I want to...

**Install the bot on Windows:**
1. Read [INSTALL_WINDOWS.md](../INSTALL_WINDOWS.md)
2. Run `INSTALL.bat`
3. If issues: See [WIZARD_TROUBLESHOOTING.md](WIZARD_TROUBLESHOOTING.md)

**Install on Termux/Android:**
1. Read [TERMUX_GUIDE.md](../TERMUX_GUIDE.md)
2. Run the one-command installer
3. If issues: See [TERMUX_TROUBLESHOOTING.md](../TERMUX_TROUBLESHOOTING.md)

**Configure the bot:**
- Edit config files using [CONFIG_DOCUMENTATION.md](CONFIG_DOCUMENTATION.md)
- Use the web dashboard at http://localhost:5000

**Troubleshoot installation:**
- Check [WIZARD_TROUBLESHOOTING.md](WIZARD_TROUBLESHOOTING.md)
- Review [ERROR_CHECKING_GUIDE.md](ERROR_CHECKING_GUIDE.md)
- See main [README.md](../README.md) troubleshooting section

**Use admin commands:**
- See [ADMIN_COMMANDS_AUDIT.md](ADMIN_COMMANDS_AUDIT.md)

**Update the bot:**
- Auto-updates are enabled by default
- Manual: `git pull` then restart

**Test my setup:**
- Run `python test_setup.py`
- Follow [TEST_PLAN.md](TEST_PLAN.md) for comprehensive testing
