# ============================================================================
# IMPORTANT: Termux/Android Installation Instructions
# ============================================================================
# The discord.py[voice] extra requires PyNaCl, which needs libsodium to build.
# On Termux, the bundled libsodium compilation fails, so you must use the
# system-installed version instead.
#
# SOLUTION (run these commands in order):
#   1. Install build dependencies:
#      pkg update && pkg upgrade
#      pkg install build-essential python binutils libsodium
#
#   2. Set environment variable:
#      export SODIUM_INSTALL=system
#
#   3. Upgrade pip and install packages:
#      pip install --upgrade pip wheel setuptools
#      pip install -r requirements.txt
#
# OR simply use the provided installation scripts which handle this automatically:
#   - bash termux_quickstart.sh
#   - bash scripts/quickinstall.sh
#   - bash scripts/install_termux.sh
# ============================================================================
